# Sayanshpal Interview Portfolio

This repo contains two portfolio projects aligned with your resume:

## 1) CRM (Customer Relationship Management) backend
- **Tech stack:** Java, Spring Boot, REST API, MySQL, Salesforce API (basic integration)
- **Folder:** `crm-backend/`
- **What it demonstrates:** CRUD APIs, sales tracking that updates inventory, and an optional Salesforce Accounts query endpoint.

## 2) GreenCart — Carbon Footprint Optimizer (Chrome Extension)
- **Tech stack:** Vanilla JavaScript, Chrome Extension MV3, Haversine formula
- **Folder:** `greencart-extension/`
- **What it demonstrates:** Computes estimated CO2 emissions using Haversine distance and suggests shipment consolidation.
